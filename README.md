# Astraeus Mechanus

Astraeus Mechanus is a 4 mechanum wheels robot, in which i aim to  improve and level up my robotics knowledge. The principal goal of this project is to have fun and be able to grasp a maximum of robotics concepts (kinematics, pid, localization, path planning etc.)

## Roadmap 

### Level 1 : Hardware & Firmware Foundation

- Define the BOM for the project
- Purchase the necessary parts for this section
 - Hardware abstraction layer :
	 - Driver for the motors [Arduino & Rasp pi]
	 - Encoders using interrupts
- Forward and Inverse Kinematics
- Closed loop with PID contol
	- Implement a PID
	- Tune the PID
- Communication using a binary serial packet structure for low latency comms

### Level 2 : Perception and Localization

- IMU integration
- Lidar integration and the map creation using slam 
- Sensor Fusion using an EKF
